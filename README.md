## Proyectos Destacados


- **[Termómetro Digital de Audiencias: Análisis de Sentimiento en YouTube con NLP avanzado](https://github.com/CristianRiquelmeF/social-listening-yt-analytics)**   
   Desarrollo de una solución de NLP orientada al análisis de audiencias digitales a partir de comentarios de YouTube. El sistema automatiza la extracción de datos mediante web scraping, procesa texto no estructurado y clasifica sentimiento con modelos Transformer en español, integrando además niveles de confianza y visualización analítica para convertir opinión pública digital en información accionable.

- **[Segmentación de Clientes con K-Modes](https://github.com/CristianRiquelmeF/Segmentacion-clientes-Kmodes/tree/main)**
   Desarrollo de un modelo de segmentación no supervisada en R para identificar perfiles de clientes a partir de variables categóricas. El valor del proyecto estuvo en incorporar un enfoque sociológico al feature engineering, transformando variables demográficas en categorías más interpretables sobre estructura familiar y momento de vida, para construir segmentos estadísticamente consistentes y analíticamente útiles.

- **[Modelo de Riesgo Crediticio con Machine Learning](https://github.com/CristianRiquelmeF/riesgo_crediticio_ml_modelos)**  
  Proyecto de clasificación binaria para predecir probabilidad de impago utilizando Python y Scikit-learn. Incluye EDA, preprocesamiento, comparación de modelos y optimización de hiperparámetros, logrando un **AUC-ROC de 0.86** con Random Forest.

- **[API de Diagnóstico con Flask, Docker y CI/CD](https://github.com/CristianRiquelmeF/api-breast-cancer)**  
  Implementación end-to-end de un modelo de clasificación médica emulada expuesto mediante API RESTful con Flask, containerizado con Docker y automatizado con GitHub Actions. Incluye pruebas, endpoint de predicción y despliegue reproducible.
